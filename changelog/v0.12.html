    # soundbot v0.12.0 "Edit Your Friends"

    Released: 08/12/2023

    Playlist editing is back! Also, you can now share playlists!

    NOTICE: If your playlist previously had a name that consisted of ONLY NUMBERS, please contact me ASAP. It may be vulnerable.

    # Major Changes

    - Brought back playlist editing, along with a few new editing commands only available to queues before. Use "/playlist edit select" to choose a playlist to edit
- Introduced new UNIQUE system for sharing playlists! Use "/playlist token get" to get a special token for a playlist of yours and get a friend to redeem it using "/playlist token redeem" to let them edit it!
- (New Commands) /cycle and /cycle-backward can now be used to cycle through the songs in a queue/playlist
- (Internal) Finalized new import system

    # Minor Changes

    - Improved performance of /playlist list massively. (For playlist of 50 items, data requests are reduced by 5x)
- Fixed any potential errors due to playlist name being too long.
- Added new caching system
- Made queues and playlistQueues share code using a generic class
- Max playlist name size increased from 63 to 64
- (Bug fix) You can now disable AutoPlay
- (Bug fix) Fixed timestamp issues
- (Internal) Removed unneeded null checks thanks to new import system.
- (Internal) Cleaned up a few files
